# 40〜50代女性のための自律神経タイプ別セルフケア診断シート

公式LINE登録後の無料特典として使える、静的な1ページWeb診断です。体のサイン・感情の揺れ・生活負荷・家族への反応・自己否定の傾向を見える化し、今日から選びやすい小さなセルフケアにつなげることを目的にしています。

## 対象ユーザー

40〜50代女性。不調や感情の揺れを性格だけの問題として抱え込みやすい方、家族に強く当たったあと自己否定しやすい方、何から整えればよいか分かりにくい方を想定しています。

## 起動方法

`index.html` をブラウザで開くと、そのまま診断を使えます。サーバーやビルド作業は不要です。

## ファイル構成

- `index.html`: ページ構造、注意書き、診断UI、CTA、参考情報
- `style.css`: スマホファーストのデザイン、配色、レイアウト
- `script.js`: 質問表示、スコア計算、結果表示、コピー機能
- `assets/moss-character.jpg`: トップページに表示する軽量化済み画像
- `README.md`: この説明ファイル

## 診断ロジック

質問は20問で、5タイプに4問ずつ紐づいています。

- よくある: 3点
- ときどきある: 2点
- あまりない: 1点
- ほとんどない: 0点

各回答の点数を紐づくタイプに加算し、最も高いタイプをメインタイプとして表示します。2番目に高いタイプはサブタイプとして表示します。最高点が同点の場合は「複数の傾向が同じくらい出ているかもしれません」と表示し、同点タイプを複数表示します。

## 医療的な注意

この診断は医療的な診断ではありません。病気の判定、医療方針の提示、不調が軽くなることの保証は行いません。強い不調が続く場合や日常生活に支障がある場合は、医療機関や専門家への相談も選択肢に入れてください。

`index.html` のCTAエリアにあるリンクを差し替えてください。

現在はオープンチャットの招待URLを設定しています。公式LINE内で配信する前提のため、公式LINEへのCTAは設置していません。

## プライバシー

名前・メールアドレス・電話番号などの個人情報入力欄はありません。回答内容はブラウザ内で計算され、外部送信は行いません。
